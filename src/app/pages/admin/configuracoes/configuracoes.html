Painel de Controle do Sistema
Gerencie as configurações globais e ações de manutenção da API Canarinho.

Configurações Gerais

Cadastro Aberto
Permitir que usuários criem novas contas livremente pelo formulário de registro.
Segurança (Rate Limiting)
Quantidade máxima de requisições que um mesmo IP pode fazer.
Tempo em que o contador de requisições será resetado para o IP.

Ações de Manutenção

Área de Risco
As ações abaixo têm impacto direto no funcionamento em tempo real do sistema. Utilize apenas se houver problemas de sincronização.
Limpeza de Cache Global
Invalida e remove todos os caches da API no backend. Isso força o banco de dados a ler tudo novamente.
